\doxysubsection{Detailed Description}
Atmel-\/supplied source file for I\+AR board startup. 

This file captures the vector table in F\+L\+A\+SH and has the required entry symbols to make the I\+AR compiler happy and generate the proper startup code to do low level initializations and then call main.
